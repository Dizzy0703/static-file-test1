WELCOME


This website is an example of a static file server built using the Express.js web development framework, which itself runs on the Node.js JavaScript runtime environment. While not nearly as powerful as a MEAN, MERN or MEVN stack, it is useful as a demonstration of Express's extremely simple static file serving capabilities, and the neat, easy-to-implement nature of plain ol' HTML and CSS.

If you want to know how it was put together, read through the other tabs of this site in left-to-right order, excluding the MongoDB page (It's extremely simple; won't take more than 20 minutes).


Node.js
Representing a 'JavaScript everywhere' paradigm that may serve to simplify web development and bring all aspects of a full stack under the control of one language and one set of developer skills.

Express.js
A server framework built upon Node.js, for creating APIs and many other forms of backend, with modules that allow it to use ORMs for MongoDB and other popular NoSQL databases.

If you are a new developer, see the pages on installing Node.js, installing MongoDB, and starting an Express server.


MEAN and Other Stacks

A MEAN stack (MongoDB, Express.js, Angular.js, Node.js) is a web software stack that implements many of the functions of the traditional LAMP stack with mostly open-source components, based (in their majority) on JavaScript and Node.js, simplifying design and making code less specific to each particular layer of the stack. The MEAN stack has several variations, including MERN (where Angular.js is replaced with React.js), and MEVN (where Angular.js is replaced with Vue.js), each of which has advantages when used with certain design architectures such as MVC and MVVM.

It should be kept in mind, however, that all of the components of this stack are readily interchangeable with layers from other web development frameworks, including ASP.NET and ASP.NET Core (which would replace the Node and Express backend) and a MySQL database (as opposed to the document-oriented mongoDB).
